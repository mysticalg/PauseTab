Privacy policy

PauseTab is local-first by design.

This policy covers the PauseTab Chrome extension, the PauseTab website, and the billing and license services that support paid plans.

What PauseTab stores locally

  • Site rules and presets
  • Daily counters and per-domain usage totals
  • Temporary passes and cooldown state
  • Appearance and weekly summary preferences

What browsing data PauseTab handles

PauseTab checks the current page URL against the site rules created by the user so it can decide whether to show a delay gate, intention prompt, budget block, session cap, or temporary pass. This matching happens in the browser and is limited to the extension's focus-control features.

What PauseTab does not collect by default

  • Full browsing history exports
  • Page text or page content
  • Passwords
  • Personal messages
  • Typed content outside PauseTab prompts

Paid plan and sync data

Paid activation can store account email, license status, Stripe customer and subscription references, activation-code hash, sync-token hash, and synced settings payloads that the user explicitly pushes to the service. The backend remains limited to billing, licensing, and sync data.

Chrome Web Store Limited Use disclosure

PauseTab uses page URL and browsing-activity data only to provide its single user-facing purpose: adding intentional friction before user-selected distracting websites and optionally syncing the rules and preferences the user chooses to sync. PauseTab does not sell this data, does not use it for advertising, and does not transfer it to third parties except when necessary to provide billing, licensing, sync, security, fraud prevention, or to comply with law.

PauseTab does not allow people to read this browsing data except when a user asks for support that requires it, or when access is required for security investigation or legal compliance.

Data use commitments

  • No sale of user data
  • No hidden tracking scripts in the extension runtime
  • No unrelated use of browsing activity
  • No remote executable code
