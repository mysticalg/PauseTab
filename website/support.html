Billing and subscription management

After checkout, the PauseTab website shows an activation code. Paste that code into the extension Plan tab to activate Pro. You can also paste the same code into the billing portal form on the home page to open Stripe's customer portal.

What to include in a support request

  • The email used at checkout
  • Your activation code, if one was issued
  • The browser and operating system version
  • A short description of the problem and what you expected
