# PauseTab PRD Summary

This repository is based on the `PauseTab — Product Requirements Document (PRD)` PDF supplied with the project.

## Product

PauseTab is a privacy-first Chrome extension that adds intentional friction before distracting websites such as YouTube, X, Reddit, Facebook, Instagram, and other user-defined domains.

## MVP scope implemented in this repo

- Manifest V3 Chrome extension
- React popup and options pages
- Service worker rule engine
- In-page intervention gate with countdown and prompts
- Rule CRUD, presets, import/export
- Global pause controls
- Daily counters and weekly summary surfaces
- Local-first storage

## Pro-ready surfaces included in this repo

- Pro feature gating in the extension
- Trial state and upgrade messaging
- Schedule, budget, session-cap, cooldown, and temporary-pass support
- Chrome sync-backed settings sync for paid/trial states
- Website and backend stubs for future checkout/license work

## Deferred production work before store launch

- Real checkout and license backend
- Final marketing domain and support email
- Web Store screenshots and promo art
- Cross-browser QA and packaging hardening
