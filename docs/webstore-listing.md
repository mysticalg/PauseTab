# Chrome Web Store Listing Draft

## Title

PauseTab — Delay Distracting Websites

## Short description

Add intentional friction before distracting websites so you browse with more control.

## Full description

### What it does

PauseTab slows down impulsive opens on distracting websites with a short countdown, optional intention prompt, budgets, and session caps.

### How it works

- Add a site you want to protect
- Pick a delay style such as Gentle, Balanced, or Strict
- When you open that site, PauseTab shows a calm intervention before the page continues

### Who it is for

- Remote workers
- Students
- Creators and developers
- Anyone trying to reduce feed-driven distraction

### Privacy promise

PauseTab is local-first by default. It does not collect page text, personal messages, or unrelated browsing analytics.

### Free vs Pro

Free includes up to 3 protected sites, delay timers, intention prompts, daily counters, popup stats, presets, and import/export.

Pro adds unlimited rules, schedules, budgets, session caps, temporary passes, weekly summaries, and sync.
