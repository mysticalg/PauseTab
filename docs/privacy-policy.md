# PauseTab Privacy Policy

## Overview

PauseTab is designed to be local-first and privacy-first. The extension exists to add intentional friction before distracting websites. It is not built to collect or monetize unrelated browsing behavior.

## Data stored locally by default

- Site rules and presets
- Daily counters and per-domain usage totals
- Temporary passes and cooldown state
- Appearance and weekly summary preferences

## Data not collected by default

- Full browsing history exports
- Page text or page content
- Passwords
- Personal messages
- Typed content outside PauseTab prompts

## Paid plan behavior

Paid or trial states can enable settings sync. In this repository, paid activation can store:

- account email
- license status
- activation code hash
- sync token hash
- synced settings payloads

The backend remains limited to billing, license, and sync data.

## Data use commitments

- No sale of user data
- No hidden tracking scripts in the extension runtime
- No unrelated use of browsing activity
- No remote executable code
